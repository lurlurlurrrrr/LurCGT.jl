# G2 Symmetry Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add core `G2` symmetry support to `LurCGT`, including the defining crystal, the two fundamental irreps, the shared alternating-product generalization required by `G2`, and dimension-regression coverage.

**Architecture:** Keep the implementation narrow by filling the symmetry hooks in `src/Symmetry/G2.jl` and exporting `G2` from `src/LurCGT.jl`, while reusing the shared tableau, crystal, and irrep-construction code. The one shared change is in `src/irep.jl`: `def_altprod` must stop assuming one-dimensional source blocks, unit norms, and unit lowering coefficients. Drive the work test-first with focused alternating-product regressions plus the broader G2 smoke and dimension checks.

**Tech Stack:** Julia, `Test`, `SparseArrays`, existing `LurCGT` symmetry/crystal infrastructure

```julia
const G2_DEF_STATES = [
    (1, (1, 1)),
    (2, (-1, 1)),
    (3, (2, 0)),
    (0, (0, 0)),
    (-3, (-2, 0)),
    (-2, (1, -1)),
    (-1, (-1, -1)),
]

function getsz_def(::Type{G2})
    return Dict(weight => (i, i) for (i, (_, weight)) in enumerate(G2_DEF_STATES))
end

getsz_def_vec(::Type{G2}) = [collect(weight) for (_, weight) in G2_DEF_STATES]
getdz(::Type{G2}, lop::Int) = lop == 1 ? (2, 0) : (-3, 1)
qlab2mwz(::Type{G2}, qlabel::NTuple{2, Int}) = (qlabel[1], qlabel[1] + 2 * qlabel[2])
getqlabel(::Type{G2}, z::NTuple{2, Int}) = (z[1], (z[2] - z[1]) ÷ 2)
```

function def_altprod(irep::Irep{S, NL, NZ, RT}, n::Int) where {S, NL, NZ, RT}
    basis_meta = get_altprod_basis_meta(irep)
    weight_dict, basis_norms = build_altprod_basis(irep, basis_meta, n)
    Sl_new = build_altprod_lowering(irep, basis_meta, weight_dict)
    inner_prod = build_altprod_innerprod(weight_dict, basis_norms)
    return Sl_new, Sz_new, mw, inner_prod
end
